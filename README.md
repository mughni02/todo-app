# Vue Todo App

Todo List application built with Vue 3 Composition API, Vite, and JSON Server.

## Features

- ✅ Add new todos
- ✅ Mark todos as complete/incomplete
- ✅ Edit todo text
- ✅ Delete todos
- ✅ Beautiful responsive UI
- ✅ API simulation with JSON Server
- ✅ Component-based architecture
